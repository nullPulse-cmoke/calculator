# 🖩 Dark Calculator

A sleek, minimal calculator with a dark UI — built with pure HTML, CSS, and JavaScript. No frameworks, no dependencies.

![Calculator Preview](preview.png)

## ✨ Features

- Clean dark theme with purple accent
- Full arithmetic: `+` `−` `×` `÷`
- Percentage and sign toggle
- Expression history display
- ⌨️ Keyboard support
- Responsive & mobile-friendly

## 🚀 Live Demo

> Open `index.html` in any browser — no build step needed.

Or use GitHub Pages:
1. Go to **Settings → Pages**
2. Set source to `main` branch / root
3. Your calculator is live at `https://yourusername.github.io/calculator`

## 🛠️ Tech Stack

| Layer | Tech |
|-------|------|
| Markup | HTML5 |
| Styling | CSS3 (Grid, Custom Properties) |
| Logic | Vanilla JavaScript |
| Fonts | Google Fonts (Space Mono + DM Sans) |

## ⌨️ Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0–9` | Input digits |
| `+ - * /` | Operators |
| `Enter` or `=` | Calculate |
| `Backspace` | Clear (AC) |
| `.` | Decimal point |
